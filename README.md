# Vedavahini
Vedavahini is an open-source repository where you can find a lot of educational and insightful content for free. The name Vedavahini means “one who carries or conveys the Vedas.” The word “Veda” is derived from the Sanskrit root vid, meaning “to know” or “knowledge.”
I'd like to inform everyone that the website’s name is for representational purposes only and the site is not intended for religious use.
The idea of this project came in my mind that present's educational system (atleast in my country) has major big flaws, such as - No flexibility - means the student has to adapt accordingly to system but it should be opposits, teachers are supposed to be trained to meet each individual student's needs.
